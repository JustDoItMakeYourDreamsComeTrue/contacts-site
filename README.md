# Todo «Список контактов»

Нативное веб-приложение на TypeScript без frontend-фреймворков.

## Что реализовано

- Верстка по подходу Mobile-First.
- SCSS + CSS-переменные.
- БЭМ-именование классов.
- Локальное подключение шрифтов `woff2` из `public/fonts`.
- CRUD контактов и групп.
- Сохранение/загрузка через `localStorage`.
- Маска телефона на базе `IMask`.
- Кастомный дропдаун (класс `CustomDropdown`) с публичными `bind` и `dataItems`.
- Попап подтверждения удаления группы с каскадным удалением контактов.
- Тостеры для создания/изменения/удаления контактов и групп.
- Проверки на дубли названий групп и телефонов контактов.
- Проверка обязательного выбора группы при создании и редактировании контакта.
- Подтверждение удаления группы с текстом из ТЗ и каскадным удалением вложенных контактов.
- ООП-классы: `ContactManager`, `StorageService`, `CustomDropdown`, `ToastManager`, `ConfirmModal`.

## Стек

- `TypeScript`
- `Vite`
- `SCSS`
- `IMask`
- `Vitest`

## Быстрый старт

```powershell
npm install
npm run dev
```

Сборка production:

```powershell
npm run build
```

Тесты:

```powershell
npm run test
```

## Деплой

Можно опубликовать на любой бесплатной платформе:

- GitHub Pages
- Vercel
- Netlify

Для GitHub Pages удобно использовать `dist/` после `npm run build`.

## Готовность к релизу

- Локально проходят: `npm run test` и `npm run build`.
- Интерфейс адаптивный, стили на `SCSS`, классы по БЭМ.
- Хранение данных выполняется через `localStorage`.
- Добавлены QoL-улучшения: более информативные тосты по операциям с группами, улучшенная читаемость длинных имен, поддержка `prefers-reduced-motion`.

## Соглашение коммитов

Используется формат коммитов по Conventional Commits с сообщениями на русском языке, например:

- `feat(ui): добавить адаптивную верстку страницы контактов`
- `feat(core): реализовать менеджер контактов и групп`
- `test(core): добавить тесты на проверки дублей`
