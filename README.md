# Todo «Список контактов»

Коротко: нативное приложение на `TypeScript` + `Vite` без frontend-фреймворков.

## По существу

- Контакты: имя, телефон, группа.
- Группы: создание, редактирование, удаление.
- Хранение данных: `localStorage`.
- Маска телефона: `IMask`.
- Тосты на ключевые действия.
- Кастомный дропдаун (`bind`, `dataItems`).

## Краткая выжимка ТЗ

- Адаптивная верстка (Mobile-First).
- `SCSS`, CSS-переменные, БЭМ.
- Локальные шрифты `woff2`.
- Запрет дублей групп и телефонов.
- Попап подтверждения удаления группы с удалением связанных контактов.

## Запуск

```powershell
npm install
npm run dev
```

## Проверка

```powershell
npm run test
npm run build
```
